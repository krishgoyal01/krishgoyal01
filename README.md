 ## 🚀 Key Projects

 ### ♟️ Chess Engine with Windows GUI (`C++` | `Win32 API` | `GDI+` | `Git`)
* **System Architecture:** Designed a modular C++ architecture with a custom procedural API, completely isolating backend game logic from the Win32/GDI+ desktop frontend to minimize component coupling.
* **Game Logic & Mechanics:** Engineered state-based move validation logic in pure C++ to track complex chess rules (castling, en passant, pawn promotion) using an efficient attack-detection matrix for checkmate verification.
* **Memory Optimization:** Optimized latency and memory footprint using C++ STL sequential containers, pass-by-reference mechanics, and stack-based memory tracking for delta changes during deep undo/redo state regressions.
* **Desktop Integration:** Interfaced directly with native Windows architecture, managing the application event loop, window handles (HWND), and message passing to synchronize inputs.

* **Operations & Visualization:** Isolated operational bottlenecks by analyzing delivery latency patterns and customer payment distributions, synthesizing multifaceted performance metrics into interactive Matplotlib visualizations.

 ### 🎮 Character Locomotion & Mechanics Prototype (`Unreal Engine` | `Blueprints`)
